  • How do I install the game? Unzip the downloaded folder and load up construct 2, once loaded up, select ‘Open Project’ and click ‘Kingdom fall.capx’ and the game will load. Click the play button on the top left of the screen and a new window should pop up with the game playing.
  • Is this game for any other platform? This is only available for the PC, since it does not support any other controllers or consoles. However, we will be looking to expand to other platforms in the future.
  • What type of game is this and is it difficult? A platformer with a challenge, traps and enemies are placed across the 3 levels.
  • Is the game played online? No, the game is not online
  • Will the game have additional levels/DLC? Yes, we will be looking to expand the game with new skins, levels and enemies as well as new game modes and features.
  • How many levels does this game have? This game has 3 levels which brings new challenges and scenery for the player.
  • Will this game be maintained and updated? Yes, thanks to Construct 2, I am able to change around events, add new events
